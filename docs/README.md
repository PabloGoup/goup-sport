## Contexto

GOUP SPORT es una plataforma de Sports Intelligence. No es una casa de apuestas, no permite apostar, no vende cuotas y no incentiva el juego.

La prediccion es una consecuencia del analisis. El producto real es la inteligencia deportiva explicable, trazable y visual.

## Uso De Estos Documentos

- Producto y negocio deben partir por la vision.
- Desarrollo debe usar el PRD y arquitectura antes de implementar nuevas funcionalidades.
- Diseno debe usar el documento de Design System y UX para mantener consistencia visual.
- Data science debe respetar la separacion entre modelos matematicos e IA explicativa.
- La ingesta de datos reales debe persistirse en base de datos antes de alimentar la UI.
